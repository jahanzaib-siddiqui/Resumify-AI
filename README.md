## 📖 About

**Resumify AI** is an intelligent resume-building platform that helps users create professional, ATS-friendly resumes with the assistance of Artificial Intelligence.

Unlike traditional resume builders, Resumify AI generates resumes based on the hiring standards, formatting, and recruitment practices of different countries, increasing the chances of passing Applicant Tracking Systems (ATS).

Whether you're applying for jobs in the **USA, Canada, UK, Germany, Australia, UAE, Pakistan**, or any other country, Resumify AI helps tailor your resume accordingly.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/resumify-ai.git
```

Go to the project folder

```bash
cd resumify-ai
```

Install dependencies

```bash
npm install
```

Install frontend dependencies

```bash
cd client
npm install
```

Install backend dependencies

```bash
cd ../server
npm install
```

Create a `.env` file

```env
PORT=5000

MONGO_URI=your_mongodb_connection

JWT_SECRET=your_secret

OPENAI_API_KEY=your_api_key
```

Run the project

```bash
npm run dev
```
